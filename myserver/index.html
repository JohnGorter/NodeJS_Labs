My Server demo
creating a live server setup

If all went well, you should be able to use your favorite editor to change this file on the filesystem and see the browser automatically refresh. The nodejs components used are:

  • express (community module)
  • socketio (community module)
  • watchdog (custom module)
  • injector (custom module)
  • myserver (custom module)
The setup works as follows: once you start the server 'node ../myserver/index.js', the server (express) is started and serves files with .html extensions. The browser is started and navigates to http://localhost:3000/index.html. Before the contents of the html page is returned, myserver uses the injector component to inject script into the resulting page. This script opens a socketio connection back to the server.

The server also starts a watcher on the filesystem for the current directory. If the file changes on the filesystem, the server instructs the connected browser (through socketio) to reload the page. The browser reloads the page and the inject takes case of reconnecting to the server with socketio.
